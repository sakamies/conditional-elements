If, Or & Else custom elements

  • name attribute checks if FormData includes an entry with that name.
  • value attribute checks if FormData includes that value
  • name & value check if FormData includes that value for that name.
  • Reacts to input & change events. Send an event manually if you need to re-evaluate the ifs in response to something other than user input.
  • If you need more complex conditions, run your condition expression in javascript and set the expression result as a value on a hidden input, like in expressiontest if clause on this page.
